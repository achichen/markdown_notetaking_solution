markdown note-taking

幾個選擇

  • nvALT
  • Noted
  • Notebooks
  • Write

nvALT

  • open source+free
  • file based
  • 很陽春但堪用
  • 但最大的缺點是
  • 很醜
  • 非常醜
  • 真的
  • 醜斃了...

Noted

  • 便宜 $4.99 for mac
  • 用自己的 db 來存 note，不太 portable
  • 可以用 notebook, favorite 來管理
  • 可以放附檔
  • sync via iCloud
  • 但沒有 mobile app...

Notebooks

  • 貴 $19.99 for Mac(有試用版)
  • NTD 390 for iphone/ipad
  • file based(可以用Dropbox等做sync)
  • 不只支援 markdown
  • text search
  • 沒有快速鍵切換 preview/edit mode

Write

  • $9.99 mac, $1.99 iPad, $1.99 iPhone
  • 目前我在用的
  • file based
  • text search
  • sync using Dropbox或其它
  • iPad app 介面很好寫
  • mac 介面也不錯

結論

  • 用 markdown 寫筆記真的很方便
  • 但是 evernote 還是有存在的價值
  • 像是 web clipping 就很有用
  • 如果 evernote 支援 markdown
  • 那我可能還是會用它當我唯一的筆記&知識庫
